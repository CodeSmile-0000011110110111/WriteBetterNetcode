Public Member Functions

 CreateDefaultSceneAttribute (string scenePath=null)
 Creates a new default scene (with Camera + Direct Light) for a unit test method. Caution: the scene contents will be deleted with the exception of the default game objects named "Main Camera" and "Directional Light". Any changes to these two objects will persist between tests. If you rename these objects they will be deleted and not restored for other tests.
