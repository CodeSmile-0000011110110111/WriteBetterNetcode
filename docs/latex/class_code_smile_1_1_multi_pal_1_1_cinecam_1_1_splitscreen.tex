Dynamically splits the screen based on number of currently joined players. \doxylink{class_code_smile_1_1_multi_pal_1_1_cinecam_1_1_splitscreen}{Splitscreen} behaviour is as follows\+: 1 player = 1 camera active 2 players = 2 cameras active (horizontal or vertical split) 3-\/4 players = 4 cameras active In 4-\/way splitscreen, if player 3 or 4 remain in the game, the screen remains 4-\/way split. If Axis is set to Horizontal\+And\+Vertical the split will be 4-\/way regardless of player count.
