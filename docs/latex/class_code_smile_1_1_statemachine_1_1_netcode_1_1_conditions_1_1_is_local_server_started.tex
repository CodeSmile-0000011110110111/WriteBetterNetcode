Called when the \doxylink{namespace_code_smile_1_1_statemachine}{Statemachine} stops, meaning it reached a State that has no transitions. 

Use this for final cleanup of the Action/\+Condition.

Before On\+Stop the On\+Exit\+State event is raised even though there is no state change, and likely no Actions/\+Conditions anyway. But in case where a \doxylink{namespace_code_smile_1_1_statemachine}{Statemachine} is manually stopped this would ensure pairing of On\+Enter\+State and On\+Exit\+State.
