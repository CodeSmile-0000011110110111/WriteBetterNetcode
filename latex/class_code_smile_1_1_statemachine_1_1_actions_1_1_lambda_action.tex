\doxysubsubsection{Detailed Description}
Generic Action that executes a lambda (System.\+Action). 

This allows for quick and dirty tests but should be avoided for production code, as using a custom \doxylink{interface_code_smile_1_1_statemachine_1_1_i_action}{IAction} implementation will provide re-\/usability and make the intent of the Action clear, specifically in logs.
