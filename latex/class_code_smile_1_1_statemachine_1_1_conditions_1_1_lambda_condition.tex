\doxysubsubsection{Detailed Description}
Generic Condition that simply executes a lambda. Provides static methods for related operators AND, OR, NOT. 

This allows for quick and dirty tests but should be avoided for production code, as using a custom \doxylink{interface_code_smile_1_1_statemachine_1_1_i_condition}{ICondition} implementation will provide re-\/usability and make the intent of the Condition clear, specifically in logs.
