\doxysubsubsection{Detailed Description}
Tests if network is fully offline -\/ the server/host is not immediately offline after calling Shutdown(). 

Technically, this checks if Network\+Manager singleton is null and if not, whether Shutdown\+In\+Progress, \doxylink{class_code_smile_1_1_statemachine_1_1_netcode_1_1_conditions_1_1_is_listening}{Is\+Listening}, Is\+Server, Is\+Host, Is\+Client are all false. It may take a few frames for all of these to reset and during this state you cannot start a new Network\+Manager session.
